## Installation Guide
1. Open npm console in project folder.
2. Do 'npm install'
3. Run 'npm start'

## Settings
1. app.js
    * Localhost
    * Application Port: 4444
2. ActiveMQ
    * Library stopmit https://www.npmjs.com/package/stompit
    * Go to producer.js and find connectionOptions= {}
        * Here you find Broker Connection Settings
